# Contributing to Awesome AI Agents 2026

Thank you for your interest in contributing!

## How to Contribute

1. Fork this repository
2. Create a new branch (`git checkout -b add-resource`)
3. Add your resource to the appropriate section
4. Commit your changes (`git commit -am 'Add new resource'`)
5. Push to the branch (`git push origin add-resource`)
6. Open a Pull Request
