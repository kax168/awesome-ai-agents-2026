## Agent Frameworks

- **[LangChain](https://github.com/langchain-ai/langchain)** - Build LLM-powered agents with memory and tools
- **[AutoGPT](https://github.com/Significant-Gravitas/AutoGPT)** - Autonomous GPT-4 agent framework
- **[CrewAI](https://github.com/joaomdmoura/crewAI)** - Multi-agent orchestration framework
- **[OpenAI Swarm](https://github.com/openai/swarm)** - Lightweight multi-agent coordination

## Coding Assistants

- **[Cursor](https://cursor.sh)** - AI-first code editor with GPT-4 integration
- **[GitHub Copilot](https://github.com/features/copilot)** - AI pair programmer by GitHub
- **[Codeium](https://codeium.com)** - Free AI code completion
- **[Aider](https://github.com/paul-gauthier/aider)** - AI pair programming in terminal

## Automation Platforms

- **[Zapier AI](https://zapier.com/ai)** - Connect 5000+ apps with AI automation
- **[Make (Integromat)](https://www.make.com)** - Visual automation platform
- **[n8n](https://github.com/n8n-io/n8n)** - Open-source workflow automation
- **[Workbeaver](https://workbeaver.com)** - Prompt-based task automation
